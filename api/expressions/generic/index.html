    Generic Expression APIs

    These expressions are available on columns of any data type.

    ValueExpr (Expr)

    Base class for a data generating expression having a fixed and known type,

    either a single value (scalar)

    case(arg)

    Create a new SimpleCaseBuilder to chain multiple if-else statements.

    Add new search expressions with the .when method. These must be comparable with this column expression. Conclude by calling .end()

    Parameters:

    Name Type Description Default
    arg None

    A value expression

    		 required

    Examples:

    >>> import ibis
>>> t = ibis.table([('string_col', 'string')], name='t')
>>> expr = t.string_col
>>> case_expr = (expr.case()
...              .when('a', 'an a')
...              .when('b', 'a b')
...              .else_('null or (not a and not b)')
...              .end())

    Methods

    approx_median(self, where=None)

    Compute the approximate median of a set of comparable values using the

    Count-Min-Sketch algorithm. Exposed in Impala using APPX_MEDIAN.

    approx_nunique(self, where=None)

    Approximate number of unique values using HyperLogLog algorithm.

    Impala offers the NDV built-in function for this.
