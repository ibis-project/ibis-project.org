ibis.expr.api.TableExpr.cross_join¶

TableExpr.cross_join(**kwargs)¶

Perform a cross join (cartesian product) amongst a list of tables, with optional set of prefixes to apply to overlapping column names

Parameters

tables (ibis.expr.types.TableExpr) –

Returns

joined

Return type

TableExpr

Examples

>>> import ibis
>>> schemas = [(name, 'int64') for name in 'abcde']
>>> a, b, c, d, e = [
...     ibis.table([(name, type)], name=name) for name, type in schemas
... ]
>>> joined1 = ibis.cross_join(a, b, c, d, e)
